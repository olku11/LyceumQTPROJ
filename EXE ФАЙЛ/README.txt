нужные библиотеки: PyQt5, sqlite3, sys
Тема проекта: Викторина
В данной программе пользователь может создавать, проходить, просматривать викторину, а также наблюдать за баллами участников
Особенность: Вопросы, у которых нет ответов, не отображаются. Система подсчета проста: 3 балла за правильный ответ и 0 за неправильный, если пользователь ввел что-то не совпадающее
с вариантами ответов, это пропускается, количество пробелов между ответами не важно
Пользователь может добавить любое количество вопросов и ответов. Возможно сделать все ответы неправильными или наоборот правильными
Есть ограничение на количество символов в ответах и вопрос
При удалении ВСЕХ ДАННЫХ пользователя спросят о его уверенности.
